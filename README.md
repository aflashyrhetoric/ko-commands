## Installation

### From Source

```bash
git clone https://github.com/aflashyrhetoric/ko.git
cd ko
go build -o ko .
```

Then move `ko` to your PATH:

```bash
mv ko /usr/local/bin/
```

## Usage

```bash
ko [command] [options]
```

Get help for any command:

```bash
ko help [command]
ko [command] --help
```
